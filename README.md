Implementation of a simple Internet Relay Chat server. 

Credits to the [University of Chicago](http://chi.cs.uchicago.edu/chirc/intro.html).

## Testing

```
cabal sandbox init
cabal install -j
chmod +x run-tests.sh
./run-tests.sh
```
